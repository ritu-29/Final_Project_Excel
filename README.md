# 📊 Sales & Customer Insights Dashboard (Excel Project)

---

## 🧾 Project Overview

This project showcases a comprehensive **Sales & Customer Insights Dashboard** built using Microsoft Excel. The dashboard transforms raw transactional data into meaningful insights, enabling analysis of customer behavior, product performance, and regional sales trends.

The primary objective is to demonstrate how Excel can be used as a powerful analytics tool for **data-driven decision-making** through interactive visualizations and advanced formulas.

---

## 🎯 Objectives

* Analyze overall **sales performance**
* Identify **high-value customers**
* Determine **most frequently purchased products**
* Evaluate **regional sales distribution**
* Build an **interactive and dynamic dashboard**

---

## 📁 Dataset Summary

The dataset consists of **100+ transactional records** including:

* Customer details (ID, Name, Segment, Since)
* Product information (Category, Name, Price)
* Transaction details (Quantity, Payment Method, Region)
* Time-based fields (Date, Timestamp)

---

## 📊 Key Metrics (KPI Summary)

* 💰 **Total Revenue:** ₹2,85,000+
* 👥 **Total Customers:** 50
* 📦 **Total Orders:** 100+
* 📈 **Average Order Value:** ₹2,800+

---

## 🛠️ Tools & Techniques Used

### 🔹 Excel Features

* Pivot Tables
* Slicers (Interactive filtering)
* Conditional Formatting
* Data Cleaning

### 🔹 Formulas & Functions

* `SUM` → Revenue calculation
* `SUMIFS` → Customer & regional aggregation
* `COUNTIF / COUNTIFS` → Product frequency
* `IF` → Logical classification
* `MATCH` → List comparison
* `TEXT` → Name abbreviation
* `RANK` → Customer ranking
* `UNIQUE` → Distinct customer count
* `Goal Seek` → Target-based analysis

---

## 📈 Dashboard Components

### 🔹 2. Product Performance Analysis

* Frequency calculated using `COUNTIF`

### 🔹 3. Regional Sales Analysis

* Sales aggregated using `SUMIFS` and Pivot Tables

### 🔹 4. Customer Segmentation

* Segments: Basic, Standard, Premium

📌 **Insights:**

### 🔹 5. Payment Method Analysis

* Distribution across Cash, Credit Card, Debit Card, PayPal

## 🔍 Advanced Analysis

### ✅ GroupBy Logic (Using SUMIFS)

Customer-wise total purchase calculated to identify high-value customers.

---

### ✅ List Comparison

Used `MATCH` and `COUNTIF` to compare datasets and identify matching values.

---

### ✅ Abbreviation Creation

Converted customer names into initials using TEXT functions for cleaner UI.

---

### ✅ Goal Seek Application

Applied Goal Seek to determine required unit price or quantity to achieve a target revenue (e.g., ₹5,000 per transaction).

## 📊 Key Insights & Storytelling

* A small group of customers drives a large portion of revenue, highlighting the importance of **customer retention strategies**
* Frequently purchased products (like appliances and accessories) ensure **steady sales volume**, while premium electronics contribute to **high revenue**
* Regional analysis shows that **West and North regions dominate sales**, making them key focus areas
* Payment behavior indicates a shift toward **cashless transactions**, especially credit cards
